# Graphiques

## Apex Chart :

[ApexCharts](https://apexcharts.com/) est une bibliothèque JavaScript moderne permettant de créer des graphiques et des visualisations interactifs avec une API facile à utiliser. Pour des instructions détaillées et des options supplémentaires, veuillez consulter la [documentation](https://apexcharts.com/docs/installation/) d'ApexCharts.

## Recharts :

[Recharts](https://recharts.org/en-US/) - Bibliothèque de graphiques repensée,2022-07-28 17:00:00 construite avec React et D3. Pour des instructions détaillées et des options supplémentaires, veuillez consulter la [documentation](https://recharts.org/en-US/guide/installation) de Recharts.

## Chart.js (Le troisième "Re Chart" est une erreur, il s'agit de Chart.js) :

[Chart.js](https://www.chartjs.org/) est une bibliothèque JavaScript gratuite et open-source pour la visualisation de données, qui prend en charge huit types de graphiques : barres, lignes, aires, secteurs, bulles, radars, polaires et nuages de points. Pour des instructions détaillées et des options supplémentaires, veuillez consulter la [documentation](https://www.chartjs.org/docs/latest/) de Chart.js.

## Unovis :

[Unovis](https://unovis.dev/) est construit avec Typescript et vous permet d'importer des modules de composants individuels pour réduire la taille de votre bundle d'application. Pour des instructions détaillées et des options supplémentaires, veuillez consulter la [documentation](https://unovis.dev/docs/intro/) d'Unovis.
