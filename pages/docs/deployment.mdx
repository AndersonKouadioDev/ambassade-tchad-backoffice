# Déploiement

Félicitations, il est temps de passer en production ! Vous pouvez déployer Next.js géré avec Vercel, ou l'auto-héberger sur un serveur Node.js, une image Docker, ou même des fichiers HTML statiques. Lorsque vous déployez en utilisant `next start`, toutes les fonctionnalités de Next.js sont prises en charge.

<Callout type="info">
  Consultez la
  [documentation officielle de Next.js](https://nextjs.org/docs/pages/building-your-application/deploying)
  pour le déploiement.
</Callout>
