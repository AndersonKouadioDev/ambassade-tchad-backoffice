# Obtenir de l'aide

Si vous rencontrez des questions ou des problèmes avec notre script, notre équipe de support est là pour vous aider. Elle répondra à vos demandes et offrira un support limité via notre système de support dédié. Pour comprendre l'étendue du support, familiarisez-vous avec la politique de support d'Envato [ici](https://themeforest.net/page/item_support_policy).

## Le support d'article inclut

1.  La disponibilité de l'auteur pour répondre aux questions.
2.  La réponse aux questions techniques sur les fonctionnalités de l'article.
3.  L'assistance pour les bugs et problèmes signalés.
4.  L'aide avec les ressources tierces incluses.

## Cependant, le support d'article n'inclut pas

1.  Les services de personnalisation.
2.  Les services d'installation.

## Accéder au portail de support

Dès l'achat de notre modèle, vous pouvez obtenir de l'aide via notre [portail de support](https://codeshaperbd.freshdesk.com/).

## Créer un ticket de support

Pour recevoir de l'aide concernant les questions liées au modèle, veuillez ouvrir un ticket de support. Assurez-vous de fournir des informations détaillées et de joindre des captures d'écran pertinentes pour une meilleure compréhension et une résolution plus rapide.

\<Callout type="info"\>
Il est possible que votre question ait déjà été traitée sur notre serveur Discord.
Si vous avez des questions sur notre modèle, n'hésitez pas à les poser là-bas
également.
\</Callout\>

## Questions relatives au framework

Le support pour les questions spécifiques au framework peut être obtenu en ouvrant une issue dans le dépôt GitHub respectif. C'est idéal pour :

1.  L'utilisation de composants spécifiques (remplacez 'x' par le nom du composant).
2.  Les problèmes où un composant ne fonctionne pas comme prévu.

## Questions relatives aux packages tiers

Si vous avez des questions concernant des packages tiers qui ne sont pas directement liés au modèle, nous vous recommandons d'ouvrir une issue dans le dépôt GitHub respectif du package. Cette méthode garantit un support direct de la part des créateurs.
