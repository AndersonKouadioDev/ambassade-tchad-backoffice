# Installation

Avant de commencer l'installation de Dashcode, assurez-vous d'avoir un dossier prêt où vous pourrez décompresser le fichier `.zip` téléchargé.

## Choisissez une version à installer :

Lorsque vous décompresserez le fichier .zip que vous avez obtenu de ThemeForest, vous verrez deux autres fichiers .zip ainsi que d'autres fichiers. L'un d'eux s'appelle la version Démo, et l'autre la version Starter :

#### 1\. Démo : `Dashcode-vX.X.X.zip`

Ce fichier `.zip` contient la version Démo, qui inclut toutes les applications, pages et sections d'interface utilisateur de l'application de démonstration.

#### 2\. Starter : `Dashcode-starter-vX.X.X.zip`

Ce fichier `.zip` contient la version Starter, qui possède toutes les parties principales et les fonctionnalités de Dashcode, mais sans les applications, pages et sections d'interface utilisateur de la Démo. Utilisez la version Starter pour construire votre application.

-----

## Démarrer l'installation :

\<Steps\>

### Étape 1

Ouvrez une fenêtre de console/terminal et accédez à votre répertoire de travail. Assurez-vous d'être dans le même dossier que le fichier `package.json` pour exécuter les commandes.

### Étape 2

`npm install` ou `yarn install`

\<Callout type="info"\>
Note : Cette commande prendra un certain temps à s'exécuter et installera toutes les
bibliothèques nécessaires dans le répertoire `node_modules`, vous permettant de commencer
le développement.
\</Callout\>

### Étape 3

Pour démarrer le projet sur votre ordinateur, configurer le serveur de développement et surveiller les modifications de code, tapez l'une des commandes suivantes :

```bash
///npm
npm run dev

///yarn
yarn dev
```

Vous pouvez visualiser le projet à l'adresse `http://localhost:3000`.

\</Steps\>
