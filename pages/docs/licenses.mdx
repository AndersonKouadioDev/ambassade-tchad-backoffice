# Licences Standard

La majorité de nos éléments (thèmes, code, graphiques, 3D et flash) sont couverts par nos Licences Standard. Si votre produit final incluant l'élément va être gratuit pour l'utilisateur final, alors une **_[Licence Régulière](https://themeforest.net/licenses/terms/regular)_** est ce dont vous avez besoin. Une **_[Licence Étendue](https://themeforest.net/licenses/terms/extended)_** est requise si l'utilisateur final doit payer pour utiliser le produit final.

|                                                            | Régulière                                   | Étendue      |
| ---------------------------------------------------------- | ------------------------------------------- | ------------ |
| Nombre de produits finaux                                  | 1                                           | 1            |
| Utilisation dans un seul produit final                     | Disponible                                  | Disponible   |
| Utilisation dans un produit final gratuit (plus d'un utilisateur final autorisé) | Disponible                                  | Disponible   |
| Utilisation dans un produit final vendu                    | Non disponible                              | Disponible   |
| Produits/services à la demande                             | Une licence par produit final personnalisé  | Non disponible |
| Utilisation dans des éléments/modèles de stock             | Non disponible                              | Non disponible |

## Note aux freelances et agences créatives

Vous pouvez facturer vos services à votre client pour créer un produit final, même sous la Licence Régulière. Mais vous ne pouvez pas utiliser une de nos Licences Standard sur plusieurs clients ou missions.
