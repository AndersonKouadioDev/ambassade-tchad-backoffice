# Nouvelle Page

Créer une nouvelle page est incroyablement simple dans Dashcode. Voici un exemple de la façon de créer votre propre page personnalisée et de la configurer selon vos besoins.

### Créer une nouvelle page

Pour créer une nouvelle page, créez un nouveau fichier `.js` ou `.jsx` dans le dossier `app`. Vous pouvez également créer une page dans un sous-dossier du dossier `app`. Par exemple, `app/foldername/page.jsx`.

```js filename="app/foldername/page.jsx" /children/
const About = () => {
  return (
    <div class="about">
      <h1>About</h1>
    </div>
  );
};
```

\<Callout type="info"\>
Vous avez créé une nouvelle page avec succès. Veuillez ajouter cette page au [menu](https://www.google.com/search?q=/docs/menu-customization).
\</Callout\>
