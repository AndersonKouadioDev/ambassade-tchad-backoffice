# Prérequis

Avant de vous lancer dans le développement avec Dashcode, assurez-vous que votre environnement de développement est équipé de Node.js et de npm ou Yarn.

##### 1\. Node.js (\>= 20.0.0)

Pour installer Node.js, rendez-vous sur [Node.js](https://nodejs.org/en) et installez la dernière version.

##### 2\. npm (\>= 5.5.1) ou Yarn (\>=1.5.1)

Pour télécharger et installer les paquets, un gestionnaire de paquets est nécessaire. Vous avez la possibilité d'utiliser [npm](https://www.npmjs.com/) ou [Yarn](https://yarnpkg.com/) comme gestionnaire de paquets. Nous préférons **Yarn** à npm. Pour vérifier l'installation de npm ou Yarn sur votre système, exécutez `npm -v` ou `yarn -v` dans une fenêtre de terminal ou de console.

```sh
//npm
npm -v

//yarn
yarn -v
```
