# Démarrage Rapide

## Obtenez Votre Accès

Dashcode est un modèle de haute qualité disponible sur ThemeForest, nécessitant l'achat d'une licence pour accéder à son code source. Après avoir acquis une licence, vous pouvez télécharger le code source directement depuis la section "Téléchargements" de votre compte ThemeForest.

-----

## Contenu du Zip

Une fois que vous avez décompressé le fichier téléchargé depuis ThemeForest, vous verrez trois dossiers : **Dashcode-vX.X.X.zip**, **Dashcode-starter-vX.X.X.zip** et **Documentation**.

### Dashcode-vX.X.X.zip

Le package **Dashcode-vX.X.X.zip** contient tous les fichiers présentés dans la [démo en direct](http://dastail.codeshaper.net/), y compris toutes les mises en page, applications, pages, composants et éléments. Vous pouvez copier les éléments nécessaires à partir de là et les utiliser dans votre projet.

\<Callout type="warning"\>
Si vous souhaitez utiliser le package Démo, il vous faudra peut-être supprimer les éléments inutilisés ou les remplacer par votre propre contenu.
\</Callout\>

-----

### Dashcode-starter-vX.X.X.zip

Le **Kit de Démarrage** est une version simplifiée de Dashcode, sans contenu de démonstration supplémentaire. Il ne contient que les éléments essentiels dont vous avez besoin pour commencer à créer votre application.

Si vous souhaitez utiliser des bibliothèques tierces avec le Kit de Démarrage, ne vous souciez pas de leur configuration. Notre modèle est très flexible. Découvrez comment démarrer avec le [Kit de Démarrage](https://www.google.com/search?q=/docs/starter-kit).

-----

### Documentation

La **Documentation** fournit des instructions claires et des conseils pour vous aider à naviguer dans Dashcode sans effort.
