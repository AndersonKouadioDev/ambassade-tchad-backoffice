# Ressources

## Questions Fréquemment Posées (FAQ)

***Des questions ?*** Nous avons compilé une liste de réponses à vos questions fréquemment posées. Si vous ne trouvez pas ce que vous cherchez ici, n'hésitez pas à nous contacter.

##### Q : Qu'est-ce qu'une Licence Régulière ?

R : Une [Licence Régulière](https://themeforest.net/licenses/terms/regular) est conçue pour un produit final unique qui est gratuit pour les utilisateurs finaux. Cette licence convient aussi bien aux projets personnels qu'aux projets clients. Notez que chaque produit final ou domaine nécessite sa propre Licence Régulière.

##### Q : Pouvez-vous expliquer la Licence Étendue ?

R : Une [Licence Étendue](https://themeforest.net/licenses/terms/extended) est nécessaire pour un produit final unique qui implique de facturer les utilisateurs finaux, par exemple avec un modèle d'abonnement ou sur une place de marché. Cela s'applique aussi bien à vos propres projets qu'à ceux développés pour des clients. Chaque produit final ou utilisation de domaine exige une Licence Étendue distincte.

##### Q : Fournissez-vous des maquettes de conception et l'accès à GitHub ?

R : Dashcode n'inclut pas automatiquement les fichiers de conception Figma, mais nous sommes heureux de les fournir gratuitement sur demande. Cela permet une personnalisation flexible dans Figma. Si vous avez besoin d'un accès GitHub, vous pouvez le demander à notre [équipe de support](https://codeshaperbd.freshdesk.com/).

##### Q : Quelles compétences sont nécessaires pour utiliser Dashcode ?

R : Pour utiliser Dashcode efficacement, une connaissance fondamentale du développement front-end est requise, en particulier avec Tailwind et React pour une personnalisation plus approfondie. Il est important de se rappeler que Dashcode est un template front-end ; une connaissance du développement back-end est également nécessaire pour un développement de projet complet.

##### Q : Comment les mises à jour sont-elles gérées ?

R : La dernière version de Dashcode peut toujours être téléchargée depuis ThemeForest sans frais supplémentaires. Les mises à jour peuvent être facilement intégrées en vous référant à notre journal des modifications (changelog) et à notre dépôt GitHub.

##### Q : Quelle est votre politique de support ?

R : Nous offrons un support dédié via notre portail, en suivant de près la [politique de support](https://themeforest.net/page/item_support_policy) d'Envato pour fournir une aide rapide et efficace.

##### Q : Dashcode vaut-il l'investissement ?

R : Absolument. Dashcode est méticuleusement conçu pour rationaliser le processus de développement, offrant un moyen fluide et efficace de construire votre interface utilisateur web. C'est un investissement rentable pour tout développeur recherchant qualité et efficacité.

##### Q : Comment puis-je obtenir de l'aide avec Dashcode ?

R : Pour obtenir de l'aide, rendez-vous sur notre [portail de support](https://codeshaperbd.freshdesk.com/), inscrivez-vous et soumettez un ticket. Notre équipe de support dédiée est prête à résoudre tous les problèmes que vous pourriez rencontrer.
